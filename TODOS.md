## Инфраструктура

### Атомарный деплой — убрать 503 на время сборки
**Что:** `scripts/build.sh` делает `rm -rf dist` перед сборкой. nginx раздаёт
`dist/` напрямую → пока сборка идёт (2-3 мин, под нагрузкой до 11 мин), прод
отдаёт 503 + maintenance-страницу.

**Зачем:** каждый деплой = многоминутный простой для всех пользователей.

**Как:** собирать в `dist.new/`, по готовности атомарно переключать симлинк
(`ln -sfn dist.new dist`). nginx тогда всегда видит полный `dist`.

**Источник:** найдено при `/qa` 2026-05-15 (прод реально ушёл в 503 на ~11 мин).

**Приоритет:** P2.

### «Нервная система» — автозадачи из бизнес-событий
**Что:** комплекс автоматики, в котором задачи создаются автоматически из других модулей:
- При новой заявке поставщику (`so_orders` INSERT) → задача создателю на отслеживание поставки.
- При истечении срока ПСЦ (`price_agreements.expires_at` за N дней) → задача владельцу прайса.
- За день до даты поставки → напоминание получателю.

**Зачем:** превращает задачи из «личного планировщика» в общий центр операционных событий. Пользователь больше не забывает.

**Цена:** 1-2 недели.

**Источник:** Roadmap-пункт «нервная система», дополнительно подсвечен в CEO-ревью 2026-05-14.

**Когда:** после того как drag-from-anywhere докажет, что связи между сущностями и задачами работают надёжно.

**Приоритет:** P2.
